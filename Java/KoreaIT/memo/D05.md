배열의 인덱스 번호
    Heap 메모리에 할당되면 이름은 존재하지 않고 주소값만 존재한다.
    따라서 시작 주소값을 배열명이 가지고 있고, 주소 연산을 통해 다음 주소값으로 이동.
    시작주소에 연산된 정수(이동한 횟수)가 인덱스 번호가 됨.
    첫번째 방은 배열명에서 0번 이동한 것과 같으므로 인덱스 번호는 0부터 시작
