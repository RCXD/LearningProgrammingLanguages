1. 자료구조의 정의
	자료들이 어떤 방식으로 저장공간 내에서 표현되고,
	그들의 관계는 어떻게 표현해 줄 것인지를 정의하는 방법

	데이터(자료)를 목적에 맞게 구조화 시켜서 저장하는 방법
	자료들이 임의의 공간에서 어떻게 구성되어 있는지를 보여주는 구조

2. 자료구조를 배워야 하는 이유
	상황에 따라서 저장해야하는 구조의 방식이 다르기 때문

	스케줄 ) 시간순 or 중요도 순
	냉장고 ) 선입선출
	엘리베이터 ) 후입선출
