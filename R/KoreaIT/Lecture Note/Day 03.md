# 정형/반정형/비정형 데이터
- 정형 데이터 : 통계청 데이터처럼 잘 정리된, 정형화된 데이터
- 반정형 데이터 : 메타정보는 있으나, 데이터가 누락된 것도 있어 연산이 어려운 데이터
- 메타정보 자체가 없는 데이터

~~~ json
{
    이름 : 코리아,
    나이 : 11,
    주소 : 서울시
}
{
    이름 : 아카데미,
    나이 : 20,
    주소 : 부산
}
{
    이름 : 정보보안
    주소 : 인천
}
~~~
위에서 이름, 나이, 주소가 **메타데이터**이다.
